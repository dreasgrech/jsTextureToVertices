Texture to Vertices

Didn't spend too much time on cross-browser testing, so if something doesn't work (including cookies, on yours, tough luck.

If your browser supports the File API, you can drag an image to the existing one to change it.
Click on the image to add vertices. Vertices are draggable, as is this dashboard.
The yellow vertex is the first vertex in the list and the blue one is the last. To delete a vertex, click on it and hit the Delete button.
Use the mouse wheel to scale.
The undo functionality is still a bit dodgy.
